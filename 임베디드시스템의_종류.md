## 1. 임베디드 시스템의 정의

임베디드 시스템(Embedded System)은 특정한 기능을 수행하기 위해 하드웨어와 소프트웨어가 하나의 장치에 내장된 컴퓨터 시스템이다. 일반적인 컴퓨터처럼 다양한 작업을 수행하는 것이 아니라, 정해진 목적을 빠르고 안정적으로 수행하도록 설계된다.

대표적인 예로 스마트폰, 자동차 전자제어장치(ECU), 전자레인지, 스마트워치, 산업용 로봇, 드론 등이 있다.

---

## 2. 임베디드 시스템을 구성하는 최소 조건

임베디드 시스템은 다음과 같은 요소로 구성된다.

- 프로세서(MCU 또는 MPU)
- 메모리(RAM, Flash Memory)
- 입력장치(센서, 버튼 등)
- 출력장치(LED, 모터, 디스플레이 등)
- 전원공급장치
- 소프트웨어(펌웨어 또는 운영체제)

이러한 구성 요소가 함께 동작하여 특정 기능을 수행한다.

---

## 3. 펌웨어, RTOS, 임베디드 OS의 용도

### 펌웨어(Firmware)

하드웨어를 직접 제어하는 기본 소프트웨어로, 장치가 정상적으로 동작하도록 한다. 주로 전자레인지, 세탁기, 리모컨 등 단순한 임베디드 시스템에서 사용된다.

### RTOS(Real-Time Operating System)

실시간 처리가 필요한 시스템에서 사용하는 운영체제이다. 여러 작업을 정해진 시간 안에 처리해야 하는 자동차, 드론, 의료기기 등에 사용된다.

### 임베디드 OS(Embedded Operating System)

임베디드 장치에서 사용하는 운영체제로, 다양한 기능과 사용자 인터페이스를 제공한다. 대표적으로 Embedded Linux와 Android 등이 있으며 스마트 TV, 키오스크, 차량 인포테인먼트 시스템 등에 사용된다.

## 5. 임베디드 시스템 활용 사례

### 통신용 네트워크 장비

공유기와 스위치는 데이터를 목적지까지 전달하는 역할을 수행한다.

### 산업용 로봇

공장에서 용접, 조립, 운반 등의 작업을 자동으로 수행한다.

### 배송용 드론

GPS와 센서를 이용하여 목적지까지 자동으로 이동하고 물품을 배송한다.

### 스마트 가전

냉장고, 세탁기, 에어컨 등은 센서를 이용하여 자동으로 동작하며 에너지 효율을 높인다.

---

## 6. 일상 속 임베디드 시스템

### 인터넷을 사용할 때

인터넷을 사용할 때는 다음과 같은 임베디드 장비를 거친다.

- 스마트폰 또는 컴퓨터
- Wi-Fi 공유기
- 모뎀
- 인터넷 서비스 제공업체(ISP)의 네트워크 장비
- 데이터센터 서버

### 교통수단 이용 시

- 버스 : 교통카드 단말기, GPS
- 지하철 : 개찰구, 발권기
- 택시 : 카드결제 단말기, 미터기, 내비게이션

### 가정에서 사용하는 제품

- 스마트 도어락
- 전자레인지
- 전기밥솥
- 세탁기
- 냉장고
- TV
- 공기청정기
- 로봇청소기

### 학교 및 회사에서 사용하는 장비

- 출입통제 시스템
- 학생증 및 사원증 카드 리더기
- CCTV
- 프린터
- 네트워크 공유기

## 7. 새롭게 만들고 싶은 임베디드 시스템

### AI 스마트 의류 관리 시스템

의류에 RFID 태그를 부착하여 옷의 위치와 보유 현황을 자동으로 관리하는 스마트 옷장을 구상하였다.


### 주요 기능

- RFID를 이용한 의류 인식
- AI 코디 추천
- 계절 및 날씨에 따른 의상 추천
- 세탁 시기 알림
- 착용 횟수 관리

### 필요한 구성 요소

#### 하드웨어

- Raspberry Pi
- RFID 리더기
- RFID 태그
- 카메라
- 터치 디스플레이
- Wi-Fi 모듈

#### 소프트웨어

- Embedded Linux
- Python
- 데이터베이스
- AI 추천 프로그램

### 기대 효과

의류를 효율적으로 관리할 수 있으며, 코디 추천과 세탁 알림 기능을 통해 사용자 편의성을 높일 수 있다.
